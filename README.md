## 🚀 Quick Start

### 🔹 Backend Setup
```bash
cd backend/HotelBooking.API
dotnet restore
dotnet run
````

### 🔹 Frontend Setup

```bash
cd frontend
npm install
npm start
```

App runs at **[http://localhost:4200](http://localhost:4200)**
