## 🚀 Overview
A modern full-stack hotel booking platform built using ASP.NET Core Web API and Angular.  
The system allows users to explore hotels, book rooms, and manage reservations, while administrators can manage hotel listings, pricing, and availability.

---

## 🛠️ Tech Stack

### Frontend
- Angular  
- TypeScript, HTML, CSS  
- RxJS  

### Backend
- ASP.NET Core Web API  
- Entity Framework Core  
- SQL Server  

### Security
- JWT Authentication  
- Role-Based Authorization  

---

## ✨ Features

### User
- Browse hotels and rooms  
- Book rooms  
- View booking history  
- Secure authentication  

### Admin
- Manage hotels  
- Manage rooms and pricing  
- Dashboard monitoring
